# StickerSmash

Installation steps: https://docs.expo.dev/get-started/installation/

Install Node.js and run script to install all dependencies. Close all apps while running and delete dependancies and retry script if fails or restart

Steps to create project: https://docs.expo.dev/get-started/create-a-project/
Steps for this project: https://docs.expo.dev/tutorial/introduction/

Download 'Expo Go' App on IOS

On Command Prompt, to create a project: npx create-expo-app my-app

(if getting this error: npm ERR! enoent ENOENT: no such file or directory, lstat 'C:\Users\roryo\AppData\Roaming\npm' Then create empty folder 'npm' on the above path to fix)

cd to 'C:\Users\roryo\my-app' or wherever project was created

To start project: npx expo start --tunnel (didnt work initially without --tunnel argument)

Scan QR code

App updates live every time code is changed and saved
